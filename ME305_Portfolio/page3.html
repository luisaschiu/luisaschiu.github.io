Below is a screenshot of the progression of plots from tuning the controller gain values. To improve the closed-loop velocity output of the motor, we changed the controller gains (also known as Kp). We did this by increasing Kp values by increments of 0.1 to obtain the smoothest results for velocity. As seen in the screenshot below, as Kp increased, the velocity values did not vary or oscillate as much over time. Therefore, through tuning, we can conclude that that a controller gain of 1 yielded the best results for an input velocity of 150 rad/s.

Below is the task diagram for lab 4. Task encoder reads motor data such as delta and uses it to calculate the measured velocity of the motor. Task motor implements the closed loop speed controller; it runs the motor using the acuation level calculated from user input values of gain and setpoint. Task user interacts with the user by prompting the user for input.
